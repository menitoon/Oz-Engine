## 𝘚𝘵𝘢𝘳𝘵𝘶𝘱 👟

In **CommandLine** type this:

 ``` pip install Oz-Engine ```
 
 Then **import** it at the top of your project like so:
 ```python 
 import OzEngine as oz 
 ```
